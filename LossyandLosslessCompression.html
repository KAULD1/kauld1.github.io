Lossy and Lossless Compression

Computer compression is removing some data to make a file smaller. The compression can lossy or lossless both having different outcomes on how the file is affected.

Lossy Compression

Lossy compression works best with images. It involves averaging out areas of colour throughout the image. This form of compression works best for images with gradients of colour rather then large plain images.

Lossless Compression

Lossless Compression works best to compress data. It involves replacing phrases and text that are repeated frequently throughout the data with symbols. When using the legend made while the document was being compressed all the data from the document can be restored to its original state.An example of this would be ZIP files as they compress the documents in this way.
